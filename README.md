# Assignment Solutions
This repository contains the solutions for the three challenges in different languages.

### Prerequisites
Below tools/technologies are used in solutions of the challenges. 
Make sure to follow the README file available under each directory to fulfil the requirements.

- [python3](https://www.python.org/downloads/) >= 3.11.1
- [pip3](https://pypi.org/project/pip/) >= 23.3.1
- [Cloud SDK](https://cloud.google.com/sdk/install) >= 414.0.0
- [Terraform](https://www.terraform.io/downloads.html) >= 1.4.0

### Code Execution
Follow the individual README file available under each solution's directory to execute the respective code.
